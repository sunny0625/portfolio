# Sukhbir Portfolio

Premium personal portfolio built with Next.js App Router, Tailwind CSS, Framer Motion, Lucide icons, and next-themes.

## Features
- Dark mode by default with light mode toggle
- Sticky navbar and smooth section navigation
- Animated hero, timeline, skills, projects, achievements, and contact blocks
- Terminal-inspired interaction section
- SEO-ready metadata and Vercel-friendly structure

## Local setup
```bash
npm install
npm run dev
```

## Deploy
Push to GitHub and import into Vercel.
